== Description ==

Suppresses feedback products in WooCommerce, no need to change anything this plugin makes itself.

Elimina los comentarios en los productos de la tienda Woocommerce, usted no necesita cambiar nada de manera manual, el plugin lo hace solo.

Requires WordPress 4.1 or higher.
